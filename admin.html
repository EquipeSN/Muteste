COMO FUNCIONA O SISTEMA

1. Você gera um código

Define o valor e a validade. O código é criptografado com o valor e prazo.

2. Você envia ao jogador

Via WhatsApp, Telegram ou qualquer canal. Use a mensagem pronta gerada.

3. Jogador aplica o código

Na plataforma → clica no saldo → "CÓDIGO ADMIN" → cola e confirma → saldo creditado imediatamente.

4. Código expira após uso ou prazo

Cada código só pode ser usado uma vez. Após usar ou expirar, fica inativo.
